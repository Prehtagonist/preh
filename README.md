## Installation & Local Development

1. Clone the repository:
```bash
git clone https://github.com/Prehtagonist/Prehtagonist-profile.git
```

2. Navigate to the project directory:
```bash
cd Prehtagonist-profile
```

3. Open `index.html` in your preferred browser to view the site locally

## Deployment

The site is ready for deployment to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- Any traditional web hosting service

### Deploy to GitHub Pages:
1. Push your code to the repository
2. Enable GitHub Pages in your repository settings
3. Select the source (usually the main branch)

### Deploy to Vercel:
1. Install Vercel CLI: `npm install -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy

## Customization

To customize the portfolio for your own use:

1. Update the content in `index.html` with your information
2. Modify the styles in `styles.css` to match your preferences
3. Adjust the JavaScript functionality in `script.js` as needed
4. Replace placeholder images with your own
